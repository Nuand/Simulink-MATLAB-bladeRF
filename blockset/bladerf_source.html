Simulink-MATLAB-bladeRF

bladerf Source block

Description

The bladerf Source block is used to receive data from a bladeRF device. The block acts as a frontend for the bladerf_source S-function.

Dialog Box

The dialog box can be used to set the parameters of the bladeRF device.

Device Tab Pane

Device Index

Specify the index of the attached bladeRF device (e.g. 0). Available devices can be found using the "Find Devices" button.

Find Devices

The Find Devices checkbox acts like a push-button and displays information and parameters about discoverd bladeRF devices in the MATLAB console. (Just to make sure, the checkbox can't be selected.)

Baseband Tab Pane

Sample Rate

Set the rate in Samples per second (Sps) of the received samples from the bladeRF device (200kHz - 40MHz) (e.g. 1e6). The bladerf Source block will report an error, if the Sample Rate isn't possible.

Low Pass Filter Bandiwdth

Specify the baseband bandwidth of the bladeRF from 1.5MHz to 28MHz(e.g. 1.5MHz).

Sample per frame

Specify the number of samples per frame of the block output (e.g. 1024).

Output Type

Choose between a frame-based or a vector-based output. See the MATLAB documentation for more information.

RF-Frontend Tab Pane

Carrier Frequency (Hz)

Specify the carrier frequency of the received signal (e.g. 100e6). Set the tunable checkbox to tune during runtime using an additional input port.

RXLNA gain (dB)

Specify any gain of the Receive Low Noise Amplifier within the gain range of the bladeRF (0-6) (e.g. 6).

RXVGA1 gain (dB)

Specify the gain of the first Receive Variable Gain Amplifier within the gain range of the bladeRF (5-30) (e.g. 15).

RXVGA2 gain (dB)

Specify the gain of the second Receive Variable Gain Amplifier within the gain range of the bladeRF (0-30) (e.g. 15).

Misc Tab Pane

Restore Block Size

Restore the geometric size of the bladerf Source block. The checkbox acts like a push-button and hence can't be set.

Display S-function parameters

Just for debug purposes: Display the S-function parameters in the MATLAB console.
